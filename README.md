# Profile Card

Flutter Web App

## Getting Started

This project is a simple profile UI based on neumorphic design.

To use this project, you need to add following dependencies to the file 'pubspec.yaml'.
- font_awesome_flutter: ^8.8.1
- google_fonts: ^1.1.0
